## Quick Start

1. **Clone the repository you'll be working on:**
   ```bash
   # For WashU Robotics
   git clone https://github.com/wustl-delivery/delivery-robo.git
   
   # For WashU GDG
   git clone https://github.com/wustl-delivery/delivery-gdg-platform.git
   ```

2. **Read the documentation:**
   - Start with the [Project Documentation](https://github.com/wustl-delivery/.github/tree/main/docs) for system overview
